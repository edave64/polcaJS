Constants

command description
PI, pi, π (half) circle constant
tau, τ (full) circle constant
E, e Euler's constant
LN10 natural logarithm of 10
LN2 natural logarithm of 2
LOG10E the logarithm of E to base 10
LOG2E the logarithm of E to base 2
SQRT1_2 square root of 1/2
SQRT2 square root of 2

Calculations

command arguments description
sin x sine of x
cos x cosine of x
tan x tangent of x
asin x arcsine of x in radians
acos x arccosine of x in radians
atan x arctangent of x in radians
atan2 x y arctangent of the quotient of its arguments in radians
log x base logarithm of x to base base
l2 x logarithm to base 2
l10 x logarithm to base 10
ln x natural logarithm (base E) of x
exp x E to the power x
ratio num, denom Ratio with num as numerator and denom as denominator. If one of the values isn't integer, the other one will be multiplied with the number of digits after the point. A ratio will automatically be shortened and most operations performed on a ratio will return a ratio themselves. num and denom have to be numbers.
factorial x calculates the factorial of x. based gamma when non-integer.
gamma x approximation of the gamma function, based on loggamma
loggamma x approximation of the natural logarithm of the gamma function
min x y minimum of x and y
max x y maximum of x and y
pow x y x to the power of y
sqrt x square root of x
ceil x rounds up x
floor x rounds down x
round x rounds x
rand, random return a random number beween 0 and 1

Operators

command arguments description
+ x y addition
- x y subtraction
* x y multiplication
/ x y division
% x y modulo
^ x y same as pow
! x same as factorial
&& x y same as min
|| x y same as max
& x y bitwise and
| x y bitwise or
++ x increment x by one
-- x decrement x by one

Loops

command arguments description
times x proc executes proc x times
forLength x proc executes proc, till the stack is bigger or equal to x

Typing

number x tries converts the given argument into a number. If this fails, it returns NaN (Not a number)
typeof x returns the type of the argument, e.g number, string or procedure
